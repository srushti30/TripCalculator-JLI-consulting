Trip Calculator 

General Information.
Calculates the cost of a vehicle driving from one point on 407ETR, to another point.

Technologies
- Java Version: openjdk-13.0.2
- JSON

Note:
Link the jar file present in file directory.
This jar file provides classes for JSON parser

Command to execute:
java -cp ".;.../file/json-simple-1.1.jar" bin/TripCalculator
java -cp ".;.../file/json-simple-1.1.jar" bin/TestTripCalculator
